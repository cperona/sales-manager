# SalesManager

Small program to test Exception handling.

It has a Sale class that contains Product arraylist, when you add calculate the total sale price it throws an EmptySaleException.
It also has the method 'indexOutOfBoundExceptionTest' to test try-catch exception handling. 

Difference between a checked exception and unchecked exceptions:
- Checked exceptions are verified at compile-time.
- Unchecked exceptions are not verified at compile-time and don't have to be declared with the 'throws' keyword.
